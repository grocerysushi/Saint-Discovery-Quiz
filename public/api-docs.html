Saint Discovery Quiz API

REST endpoints for accessing the saints database, feast day calendar, and AI-powered patron saint matching.

https://saintdiscoveryquiz.com

Endpoints

  • GET/api/health
  • GET/api/saint-of-the-day
  • GET/api/feast-day
  • GET/api/saints
  • GET/api/saints/:name
  • POST/api/ai-match
GET /api/health

Returns server health status. Use for uptime monitoring.

Response

JSON
{
  "status": "ok"
}
GET /api/saint-of-the-day

Returns today's featured saint. Matches by actual feast day when possible; falls back to a deterministic daily rotation.

Response

JSON
{
  "date": "Tuesday, February 25, 2026",
  "dayOfYear": 56,
  "saint": {
    "name": "St. Walburga",
    "feastDay": "February 25",
    "knownFor": "English missionary to Germany...",
    "patronOf": "plague, famine, storms",
    "dates": "c. 710–779",
    "origin": "England/Germany",
    "gender": "Female",
    "quote": "..."
  }
}
GET /api/feast-day

Returns all saints whose feast day matches the given date. Defaults to today if no date provided.

Query Parameters

ParamTypeDescription
dateoptional string Date in YYYY-MM-DD format. Defaults to today.

Response

JSON
{
  "date": "Saturday, June 29, 2024",
  "feastDay": "June 29",
  "count": 2,
  "saints": [
    {
      "name": "St. Peter",
      "feastDay": "June 29",
      "knownFor": "First Pope, leader of the apostles...",
      ...
    }
  ]
}
GET /api/saints

Paginated, filterable saints directory. Returns all saints by default.

Query Parameters

ParamTypeDescription
genderoptional string Male or Female
traitoptional string Filter by trait, e.g. courage, humility
searchoptional string Text search across name, knownFor, and patronOf
limitoptional integer Max results to return (1–200). Defaults to all.
offsetoptional integer Number of results to skip. Defaults to 0.
fieldsoptional string Comma-separated field names for projection, e.g. name,gender,feastDay. Available: name, feastDay, knownFor, patronOf, dates, origin, gender, traits.

Response

JSON
{
  "total": 191,
  "limit": 3,
  "offset": 0,
  "count": 3,
  "saints": [
    { "name": "St. Mary", "gender": "Female", "feastDay": "August 15" },
    ...
  ]
}
GET /api/saints/:name

Returns a single saint by URL-friendly slug name. The slug is the saint's name lowercased with spaces replaced by hyphens.

Path Parameters

ParamTypeDescription
namerequired string URL slug, e.g. st-thomas-aquinas

Response

JSON
{
  "name": "St. Thomas Aquinas",
  "feastDay": "January 28",
  "knownFor": "Doctor of the Church, wrote Summa Theologica...",
  "patronOf": "Students, universities, theologians",
  "dates": "1225–1274",
  "origin": "Italy",
  "gender": "Male",
  "traits": ["intellectual", "wisdom", "teaching", ...],
  "quotes": ["To one who has faith, no explanation is necessary..."]
}
POST /api/ai-match

AI-enhanced patron saint matching powered by GPT-4o mini. Analyzes quiz responses against top candidate saints and returns a personalized match with explanation.

Requires configuration. Returns 503 if OPENAI_API_KEY is not set. The quiz client falls back to algorithmic matching automatically.

Request Body

FieldTypeDescription
userAnswersrequired array Array of answer objects, each with a traits array of strings.
topCandidatesrequired array Array of candidate objects with saint (object with name, dates, knownFor, patronOf) and score (number).
userGenderrequired string Male or Female

Example Request

JSON Body
{
  "userAnswers": [
    { "traits": ["courage", "faith"] },
    { "traits": ["leadership", "dedication"] }
  ],
  "topCandidates": [
    {
      "saint": {
        "name": "St. Joan of Arc",
        "dates": "1412–1431",
        "knownFor": "Led French army, martyred for faith",
        "patronOf": "soldiers, France"
      },
      "score": 42.5
    }
  ],
  "userGender": "Female"
}

Response

JSON
{
  "success": true,
  "saintName": "St. Joan of Arc",
  "explanation": "Your courageous spirit and deep faith...",
  "inspiration": "Let Joan's fearless trust in God inspire...",
  "fullResponse": "SAINT: St. Joan of Arc\nEXPLANATION: ..."
}
